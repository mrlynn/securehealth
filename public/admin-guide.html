ADMIN Dashboard Guide

Complete guide to using the SecureHealth Admin Dashboard for monitoring and managing the system.

Admin Access
How to Access Admin Dashboard:
  1. Log in with an admin account:
    • Email: admin@securehealth.com
    • Password: admin123
  2. Look for the "Admin" button in the navigation bar
  3. Click "Admin" to access the dashboard
Note: Only users with the isAdmin flag set to true can access the admin dashboard. Non-admin users will be redirected to the patients page.
Dashboard Metrics

View key performance indicators:

  • Total audit events
  • Last 24 hours activity
  • Patient count overview
Audit Logs

Monitor system activity:

  • Real-time audit events
  • Filter by action type
  • Filter by entity type
  • Adjust result limits
Admin Features
Audit Log Monitoring

Track all user actions including:

  • Patient data access (read operations)
  • Patient record creation
  • Patient record updates
  • Patient record deletions
  • User login/logout events
Advanced Filtering

Filter audit logs by:

  • Action: create, read, update, delete
  • Entity: patient, user, system
  • Limit: 25, 50, or 100 recent events
  • Time range: Last 24 hours metrics
Security Features
Access Control
  • Admin-only dashboard access
  • Automatic redirect for non-admin users
  • Session-based authentication
  • Secure API endpoints
Audit Trail
  • Complete action logging
  • IP address tracking
  • Timestamp recording
  • User identification
Quick Start
Getting Started with Admin Dashboard:
  1. Login: Use admin@securehealth.com / admin123
  2. Navigate: Click "Admin" in the navigation bar
  3. Monitor: View real-time audit metrics
  4. Filter: Use dropdowns to filter audit logs
  5. Refresh: Click refresh button for latest data
