How to Use

Add the widget to any page with simple HTML:

1. Include the widget script:
<script src="/assets/js/github-widget.js"></script>
2. Add the widget HTML:
<div data-securehealth-github-widget data-repo-owner="mrlynn" data-repo-name="securehealth" data-theme="light"> </div>
3. Or use JavaScript:
const widget = new SecureHealthGitHubWidget({ repoOwner: 'mrlynn', repoName: 'securehealth', container: '#my-container', theme: 'light', showStats: true, showButtons: true });
Configuration Options:
  • repoOwner - GitHub username/organization
  • repoName - Repository name
  • container - DOM element or selector
  • theme - 'light' or 'dark'
  • showStats - Show/hide GitHub stats
  • showButtons - Show/hide action buttons

Event Handling

The widget fires custom events you can listen to:

document.addEventListener('securehealth-github-interaction', function(event) { console.log('GitHub interaction:', event.detail.action); // Handle the interaction });
